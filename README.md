# BeatSaber_OffsetFixer

Plugin features:
- Reaction Time (ms)
- NJS multiplier (value / NJS)
- Snap Offset to grid via Precision (based on beat duration)
- Automatic RT overwrite on Speed Modifier (SS, FS, SF) 
  
Configuration available in the Gameplay mod tab.
  
Plugin Requirement:
- BS_Utils
- BSML
